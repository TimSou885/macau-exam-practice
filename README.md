# 澳門統考練習 | 430 / 260 綜合能力評估

Micro SaaS 網頁應用，用於澳門政府 **430 統考**（學士學位程度綜合能力評估）與 **260 統考**（技術員程度綜合能力評估）題目練習。

## 功能

- **430 / 260 統考**：選擇考試類型後進入練習
- **題目分類**：法律、數學、圖表分析、推理邏輯、閱讀理解
- **練習流程**：逐題作答、即時顯示正確答案與解析
- **錯題本**：答錯題目自動加入錯題本，可集中複習或移除
- **結果頁**：答對題數、得分、是否達 50 分合格參考
- **進度儲存**：錯題與作答統計存於瀏覽器 localStorage

## 技術

- React 18 + Vite
- React Router 6
- Tailwind CSS
- 題目資料為前端 JSON，可擴充為 API 或更多題庫

## 開發

```bash
npm install
npm run dev
```

瀏覽器開啟 `http://localhost:5173`。

## 建置

```bash
npm run build
npm run preview
```

## 部署

- **Vercel（推薦）**：匯入 GitHub repo 後一鍵部署，已含 SPA 路由設定（`vercel.json`）。詳見 [DEPLOY.md](./DEPLOY.md)。
- **Supabase Storage**：執行 `npm run build:supabase` 產出 Hash 路由版，將 `dist/` 上傳至 Supabase 公開 bucket 即可。詳見 [DEPLOY.md](./DEPLOY.md)。
